# Raw Corpus

Place your raw source material here before running the distillation pipeline.

## Recommended Structure

```
_raw/
├── social/          # Social media exports (X, 即刻, Weibo, etc.)
├── writing/         # Blog posts, newsletter archives, articles
├── interviews/      # Podcast/interview transcripts
├── chat/            # Chat logs (with permission)
└── other/           # Books, slides, emails, etc.
```

## Tips

- **More is better at input.** The distillation pipeline filters; you don't need to pre-select.
- **Preserve original format** where possible — formatting reveals how someone organizes thought.
- **Speaker diarization matters** for interviews — label who said what.
- **Always get explicit permission** for private communications.

See [distillation-guide.md](../../../docs/distillation-guide.md) for the full collection guide.
