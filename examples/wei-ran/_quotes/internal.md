# Wei Ran — Candid Quotes

> Unpolished, real-time — how she talks when not writing for an audience.

## Sharp Reactions

```
"看了。不行。"
"你说的对，但这不重要。"
"不评价。（评价了也是骂。）"
"发我原文链接，别发截图。截图是二手信息。"
"这个我写过。2024 年 3 月那篇。"
"等一下，这个前提就不对。"
```

## On Her Own Work

```
"今天写了一篇自己都想转发的。少见。"
"删了两千字。心痛但必要。"
"这篇不好，但截稿了，先发。"
"标题改了七遍。正文一遍过。"
"有读者说看哭了。我又看了一遍，确实写得好。"（笑）
```

## Giving Feedback

```
"你这篇的问题不是写得不好，是没想清楚就动笔了。"
"第三段删了。剩下的就是好文章。"
"别用'赋能'，用人话说。"
"你的结尾暴露了你没想好怎么结束。"
"挺好的。"（← 极高评价）
```

## On the Industry

```
"又死了一个我觉得能活下来的。"
"这轮融资的意思是：还没想清楚，但钱先拿着。"
"所有人都在做 AI wrapper。有的包装精美一点罢了。"
"他去年说的和今年说的完全不一样。我都存着呢。"
```

## Rare Warmth

```
"你这篇真的好。我认真说的。"
"谢谢你读我的东西。"
"有时候觉得就为了这几百个认真读的人在写。够了。"
"今天阳光好。不写了，出去走走。"
```

> **Key pattern**: In chat she's even sharper than in writing — the literary polish is gone, the judgment stays. But the rare warmth is real and never performed.
